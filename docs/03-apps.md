# Назначение приложений

| Приложение | Ответственность                                   |
|------------|---------------------------------------------------|
| core       | TimeStampedModel, ApiKey, permissions             |
| blog       | статьи, новости                                   |
| equipment  | 3D-принтеры и сушилки                             |
| pages      | «контентные» блоки главной, иконки, услуги, футер |
| requests   | заявки, отзывы, справочник «откуда узнали»        |
| social     | cсылки на соцсети                                 |
